Material Requerido

Os seguintes materiais serão necessários para montarmos o nosso projeto

  • 1 x Placa Arduino Uno R3 com cabo USB
  • 1 x Protoboard
  • 1 x LED vermelho
  • 1 x Resistor 220 ohm
  • 2 x Jumpers macho-macho
Protoboard

Peça fundamental em todos os projetos de prototipagem eletrônica, a Protoboard é o local onde os componentes eletrônicos são montados, utilizando trilhas previamente conectadas.

LED

O LED (Light Emitting Diode) é um diodo emisor de luz. Uma espécie de mini-lâmpada, de baixo consumo.

É importante frisar que ele possui polaridade, exigindo cuidado ao ser incluído em um circuito. Se for ligado incorretamente, será danificado.

Resistor

O resistor é um componente eletrônico discreto cujo propósito é limitar o fluxo de corrente entre dois pontos de um circuito.

Jumper

Os Jumpers são fios, ligações simples, usados para conectar pontos diferentes dos circuitos.

Circuitos

São conjuntos de componentes eletrônicos organizados de forma que o fluxo da corrente percorra o caminho fechado entre o ponto positivo e o negativo da fonte energia.

Montagem

Perceba que o pino 11 está conectado diretamente ao pólo positivo do LED, enquando o GND (terra), passa pelo resistor, em direção ao pólo negativo.

Esta configuração é importante, para evitar danificar o equipamento. Como mencionado antes, o LED é um dispositivo com polaridade, de modo que, se os fios estiverem em posição inversa ao da figura, ele vai queimar.
